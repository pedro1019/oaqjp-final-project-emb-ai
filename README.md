# Final project

Proyecto Final — Desarrollo y Despliegue de Aplicaciones Web Basadas en IA  
Autor: Pedro vargas  
Fecha: Julio 2026  

## Descripción
Esta aplicación utiliza la biblioteca Watson NLP para detectar emociones en texto.  
Incluye un servidor Flask que expone un endpoint `/emotionDetector` para recibir texto y devolver las emociones detectadas.
